# ELS-02 SysEx Manager Pro

ELS-02（Electone）向けの SysEx を GUI で扱うためのデスクトップアプリです。  
MIDI 入出力ポートを選択し、受信した SysEx から音色データを登録・保存し、任意のセクションへ送信できます。

## 主な機能

- MIDI 入力 / 出力ポートの選択
- SysEx 受信ログの表示
- **Voice Index** 画面で音色を登録（SysEx 待受）
- 登録した音色を `voice.json` に保存
- 登録済み音色をワンクリックで UK1 へ送信
- セクション（UK/LK/PK/Lead）指定送信用ロジックを内蔵

## 動作環境

- Python 3.12 以上
- MIDI I/O が利用可能な環境（例: rtmidi 対応）

## セットアップ

このプロジェクトは `uv` を使ったセットアップを想定しています。

```bash
uv sync
```

`pip` を使う場合は以下でも実行可能です。

```bash
pip install -e .
```

## 起動方法

```bash
python main.py
```

## 使い方

1. アプリ起動後、左サイドバーで MIDI の Input / Output ポートを選択します。
2. 必要に応じて機器から SysEx を送信し、ログで受信を確認します。
3. 「音色インデックス」を開き、「+ Add New Voice (Wait SysEx)」を押します。
4. 10 秒以内に対象 SysEx を受信すると、音色名入力ダイアログが表示されます。
5. 名前を入力すると `voice.json` に保存され、一覧ボタンから再送信できます。

## 保存ファイル

- `voice.json`: 登録した音色ライブラリ（音色名と 6byte データ）

## 注意事項

- SysEx 受信判定は特定フォーマット（コード内条件）を前提にしています。
- Output ポート未接続時は送信できません。
- 環境によっては `python-rtmidi` のビルド/導入に追加設定が必要な場合があります。
